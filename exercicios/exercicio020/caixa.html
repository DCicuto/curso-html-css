Modelo de caixa padrão

Os elementos HTML são exibidos como caixas retangulares. Cada caixa tem uma largura e uma altura, e pode ter margens, preenchimento e bordas.

Modelo de caixa em linha

Os elementos HTML são exibidos como caixas retangulares, mas como se fossem palavras em um texto.

Modelo de caixa em linha em bloco

Os elementos HTML são exibidos como caixas retangulares, como se fossem parágrafos em um texto.

Modelo de caixa absoluta

Os elementos HTML são exibidos como caixas retangulares, como se fossem absolutos em relação ao seu contêiner.

Modelo de caixa fixo

Os elementos HTML são exibidos como caixas retangulares, como se fossem fixos em relação à janela do navegador.

Modelo de caixa relativo

Os elementos HTML são exibidos como caixas retangulares, como se fossem relativos à sua posição normal.

Modelo de caixa estático

Os elementos HTML são exibidos como caixas retangulares, como se fossem estáticos em relação ao seu contêiner.
