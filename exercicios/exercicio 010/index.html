Usando links

1 - Links externos

Para criar um hyperlink, devemos criar âncoras através da tag . O principal atributo dessa tag é o href, que cria uma referência hipertexto. Vamos ver um exemplo simples:

Você pode acessar meu repositório no GitHub

 Acesse meu perfil no Git

Por padrão, sempre que um visitante clique em um hyperlink, o site de destino abre na mesma janela do site que continha esse link. Ou seja, o conteúdo anterior vai deixar de ser exibido para mostrar o novo conteúdo. Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target:

Usando o target na apresentaçao do site não muda nada, mas ao clicar no link vai ser aberto uma nova aba e nào sobrepor a aba já aberta :

Você pode acessar meu repositório no GitHub

Acesse meu perfil no Git

Observação:

Existe um recurso bem interessante para links que é indicar qual é a natureza do destino usando o atributo rel. Esse atributo aceita vários valores, entre eles vou citar:

  • next indica que o link é para a próxima parte do documento atual
  • prev indica que o link é para a parte anterior do documento atual
  • author indica que é um link para o site do autor do artigo atual
  • external indica que é um link para outro site que não faz parte do site

No meu exemplo ultilizei o external

2 - Links internos

Link interno, vai fazer a relação a outra pagina que está dentro do nosso servidor e diz respieto a nossa propria pagina. Para criar um link interno, devemos usar o mesmo procedimento que usamos para criar um link externo, mas com uma diferença: o valor do atributo href deve ser o ID do elemento que queremos linkar. Vamos ver um exemplo:
