Trabalhando com listas

1 - Listas ordenadas

A HTML chama de ordered lists todas aquelas listas onde a ordem dos itens é algo muito importante. Um passo-a-passo para criar um bolo, uma lista de aprovados no vestibular e uma lista com os carros mais caros do mundo são exemplos de listas ordenadas.

Para criar uma ordered list, vamos usar a tag ol para delimitar a lista e li (list item) para identificar cada item da lista. Exemplo:

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

Observação: A tag "ol" possui o parâmetro type, onde podemos configurar o tipo de marcador da lista, como numérico, alfabético, romano, etc. O valor padrão é numérico, abaixo exemplifico:

Com o valor "1" Cria listas numeradas. Ex: 1, 2, 3, 4,

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

Com o valor "A" Cria listas alfabéticas em maiúsculas. Ex: A, B, C, D

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

Com o valor "a" Cria listas alfabéticas em minúsculas. Ex: a, b, c, d

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

Com o valor "I" Cria listas romanas em maiúsculas. Ex: I, II, III, IV

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

Com o valor "i" Cria listas com algarismos romanos em minúsculas. Ex: i, ii, iii, iv

  1. Acordar 1
  2. Trocar Maria Luz
  3. Tomar café 2
  4. lavar o rosto 3
  5. Reunião 4

2 - Listas Não ordenadas

A HTML chama de unordered lists todas aquelas listas onde a ordem dos itens não é importante. Uma lista de compras, uma lista de tarefas a fazer e uma lista de presentes são exemplos de listas não ordenadas.

E Para criar uma unordered list, vamos usar a tag "ul" para delimitar a lista e a tag "li" para criar cada um dos seus itens internos.

  • Arroz
  • Feijão
  • Macarrão
  • Suco

Vemos que o marcador padrão para as "ul" é a bolinha, mas assim como nas listas ordenas dentro das "ol", aqui também podemos configurar o tipo de marcador da nossa lista, abaixo vemos todas as formas, que são apenas 3 kkkkkk

Com o valor "disc" Cria listas com marcadores circulares.

  • Arroz
  • Feijão
  • Macarrão
  • Suco

Com o valor "circle" Cria listas com marcadores circulares vazados.

  • Arroz
  • Feijão
  • Macarrão
  • Suco

Com o valor "square" Cria listas com marcadores quadrados.

  • Arroz
  • Feijão
  • Macarrão
  • Suco

    4 - Listas de definição:

    As listas de definição são como por exemplo um dicionário, temos os termos e as suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos.

    Toda lista de definições está dentro de uma tag "dl" "/dl" (definition list) Cada termo é um "dt" (definition term) e cada descrição é um "dd" (definition description). Assim como os itens da lista, essas duas últimas tags possuem fechamento opcional, segundo a referência oficial da HTML5.
