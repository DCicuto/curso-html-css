Trabalhando com fontes

No código acima, ele vai usar preferencialmente a fonte Arial. Caso a fonte Arial não seja encontrada no aparelho do visitante, o navegador vai selecionar a fonte Arial normal. Em último caso, se tudo der errado, o sistema selecionará uma fonte genérica sem serifa.

SEQUÊNCIAS SEGURAS: Existem as chamadas sequências seguras para especificações de famílias de fontes. Para ver quais são elas, abra o Google e faça uma rápida busca por CSS Web Safe Font Combinations.
