Alinhamento em CSS

Sbtitulo com alinhamento padrão à esquerda.

Paragrafo com alinhado à direita.

Esse parágrafo tem o alinhamento justificado. Esse tipo de alinhamento é bastante utilizado para textos longos e grandes parágrafos, para dar uma sensação de simetria. O alinhamento justificado não é recomendado para textos curtos, pois pode deixar o texto com espaçamento desigual entre as palavras, o que pode dificultar a leitura. IMPORTANTE: Nunca alinhe texto em HTML, sempre em CSS, alinhamento em HTML é considerado obsoleto, e pode causar problemas de acessibilidade e responsividade. HTML é SEMÂNTICO, CSS é ESTILÍSTICO.
